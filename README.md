# System Performance Diagnostics

A Java-based system performance diagnostics tool that analyzes system log files and generates structured performance reports.

## Features
- Parses system logs (CSV format)
- Categorizes issues by type
- Analyzes performance patterns
- Generates diagnostic summaries

## Technologies Used
- Java
- OOP design principles
- File I/O
- Data parsing and processing

## How to Run
1. Compile the project
2. Run Main.java
3. Provide logs.csv as input
